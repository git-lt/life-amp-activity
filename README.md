# 春节温暖回家路 临摹

H5 营销活动页面 一镜到底 效果临摹

源代码来自 [码云仓库 pixi h5 页面](https://gitee.com/wuchunling/pixiJS)

## 优化

- 代码组件结构优化
- 使用了 `ES6` 语法重写逻辑
- 数据资源配置化
